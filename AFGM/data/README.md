# Data

This repository does not redistribute original or processed image files.

## CUB-200-2011

The CUB-200-2011 foreground-segmented inputs used in the computational experiments should be generated following the preprocessing procedure described in the HVE framework:

https://github.com/gyhandy/Humanoid-Vision-Engine

Please obtain the original CUB-200-2011 dataset from the official dataset source and then follow the HVE preprocessing pipeline.
